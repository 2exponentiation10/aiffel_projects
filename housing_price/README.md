## 회고
하이퍼 파라미터를 튜닝하는데 시간이 너무 오래걸려 시간내에 완성하지 못한것이 조금 아쉬웠다. 다음부터는 로컬에서 실행해보는게 좋을것같다.
그리고 피쳐도 만져보멶서 하고 싶었는데 이또한 시간이 부족해 기본적인 세팅정도밖에 하지 못하고 시간내로 테스트 해보지 못한 부분들이 아쉬웠다.
코드에대한 전반적인 리딩은 되는데 그걸 활용하는것이 아직은 조금 어렵게 느껴져 더 공부해야할것같다.
